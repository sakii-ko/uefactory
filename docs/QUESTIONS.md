# 问题与决策请求(QUESTIONS)

> Coder 遇到 PLAN/CONVENTIONS 没覆盖的决策点时写在这里,然后**继续做不被阻塞的任务**。
> Planner 会在每轮 review 时批复(直接在条目下写 `ANSWER:`);重大问题升级给 Owner。
> 已解决的条目移到文末「已归档」。

## 条目模板

```markdown
### Q<编号> [YYYY-MM-DD] <一句话问题>
- 背景:
- 备选方案:A / B(各自代价)
- 我的倾向:
- 阻塞哪些任务:(无则写"不阻塞")
```
